About this Website

FPHP is a newly established web site that encourages learning some good practices of PHP in order to learn common web vulnerabilities and protect against it. It combines vulnerabilities in a sort of challenges where you need to exploit the vulnerability to find a "flag", which shows that you successfully know the miscofiguration.
